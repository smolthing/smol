## Getting Started

Install the latest version of [Wasp](https://wasp.sh/) or run the command:
```bash
curl -sSL https://get.wasp-lang.dev/installer.sh | sh
```

Create a new SaaS app with the following command:

```bash
wasp new -t saas
```
